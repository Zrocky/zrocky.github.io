Mac设置Git代理

设置git代理：

  1. 设置github.com走代理

    编辑~/.gitconfig文件，添加以下代码：

    [https "https://github.com"]
  proxy = socks5://127.0.0.1:1086

  2. 设置github.com走IP（效果未验证，可跳过此步骤）

    编辑/etc/hosts文件，添加以下代码：

    151.101.72.249 http://global-ssl.fastly.Net
192.30.253.112 http://github.com

查看git当前代理设置：**

git config -l

如果出现箭头所指代理，即设置成功
